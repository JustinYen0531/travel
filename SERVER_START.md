# 佛光大學互動報告：啟動指南 🚀

這個資料夾包含你為這堂課準備的所有素材。要開始你的 15 分鐘報告，請參考以下步驟：

---

## 🏗️ 方式一：直接開啟 (推薦)
1. 在檔案總管裡對著 `佛光全紀錄網頁版_互動遊戲.html` 點擊滑鼠右鍵。
2. 選擇「開啟檔案」 -> 「Google Chrome」或「Microsoft Edge」。
3. 這是最保險的做法，適合在學校電腦上直接操作。

---

## 🏗️ 方式二：使用本地服務器 (開發者推薦)
如果你想在 VS Code 裡同步預覽，請在終端機 (Terminal) 中執行：

```powershell
python -m http.server 8080
```

接著在瀏覽器打開：`http://localhost:8080/佛光全紀錄網頁版_互動遊戲.html`

---

## 📸 關於照片格式的提醒
由於你的原始照片是 Apple 的 `.HEIC` 格式，**瀏覽器預設是無法直接顯示的**。
- **報告建議**：在展示互動網頁時，專注於「文字冒險與決策」。
- **輔助展示**：當講到特定的視覺衝擊（如雲起樓的末日感）時，可以直接用 Windows 內建的「相片」軟體滑動播放該照片編號（如 `IMG_0671`）來進行輔助說明。
